leetcode 5853. 从子集的和还原数组

发表于 2021-08-23 | 更新于 2021-08-30 | 分类于 leetcode题目 | 评论数：

题目：

存在一个未知数组需要你进行还原，给你一个整数 n 表示该数组的长度。另给你一个数组 sums ，由未知数组中全部 2n 个 子集的和 组成（子集中的元素没有特定的顺序）。

返回一个长度为 n 的数组 ans 表示还原得到的未知数组。如果存在 多种 答案，只需返回其中 任意一个 。

如果可以由数组 arr 删除部分元素（也可能不删除或全删除）得到数组 sub ，那么数组 sub 就是数组 arr 的一个 子集 。sub 的元素之和就是 arr 的一个 子集的和 。一个空数组的元素之和为 0 。

注意：生成的测试用例将保证至少存在一个正确答案。

示例：

示例 1：

输入：n = 3, sums = [-3,-2,-1,0,0,1,2,3]
输出：[1,2,-3]
解释：[1,2,-3] 能够满足给出的子集的和：
- []：和是 0
- [1]：和是 1
- [2]：和是 2
- [1,2]：和是 3
- [-3]：和是 -3
- [1,-3]：和是 -2
- [2,-3]：和是 -1
- [1,2,-3]：和是 0
注意，[1,2,-3] 的任何排列和 [-1,-2,3] 的任何排列都会被视作正确答案。

示例 2：

输入：n = 2, sums = [0,0,0,0]
输出：[0,0]
解释：唯一的正确答案是 [0,0] 。

示例 3：

输入：n = 4, sums = [0,0,5,5,4,-1,4,9,9,-1,4,3,4,8,3,8]
输出：[0,-1,4,5]
解释：[0,-1,4,5] 能够满足给出的子集的和。

提示：

  • 1 <= n <= 15
  • sums.length == 2^n
  • -104 <= sums[i] <= 10^4

分析：

可以通过最小的两个数的差得出原始数组中其中一个数字的绝对值
原理：最小的数是所有负数之和。第二小的数要么是最小的数加上最小的正数，要么是最小的数减去最大的负数

这样得出一个数后，可以将sums中数分为两组，一组为包含这个数的，一组为不包含这个数的

取其中小的一组（应该可以随意选择哪组），继续上面的步骤，直到n个数全部得出

具体讲解：https://www.bilibili.com/video/BV1Qh411i7tS?p=5

代码：

class Solution {
public:
    vector<int> recoverArray(int n, vector<int>& sums) {
        sort(sums.begin(), sums.end());
        int nn = n;
        int lr = sums.front() + sums.back();
        vector<int> ret;
        while(nn > 0){
            vector<int> s;
            int dif = sums[1] - sums[0];
            ret.push_back(dif);
            unordered_map<int,int> m;
            for(int i = 0; i < sums.size(); i++) m[sums[i]]++;
            for(int i = 0; i < sums.size(); i++){
                if(m.count(sums[i])){
                    s.push_back(sums[i]);
                    if(!--m[sums[i]]) m.erase(sums[i]);
                    if(!--m[sums[i] + dif]) m.erase(sums[i] + dif);
                }
            }
            sums = s;
            nn--;
        }

        for(int i = 0; i < (1<<n); i++){
            int sum = 0;
            for(int j = 0; j < n; j++){
                sum += (i & (1 << j)) ? ret[j] : -ret[j];
            }
            if(sum == lr){
                for(int j = 0; j < n; j++){
                    if(!(i & (1 << j))) ret[j] *= -1;
                }
                return ret;
            }
        }
        return ret;

    }
};
